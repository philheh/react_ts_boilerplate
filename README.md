# react_ts_boilerplate
Wrap up your ts/react/webpack game with a simple preconfigured boilerplate 

### Steps:

1. Clone this repo


2. Install Deps
    ```bash
   npm ci
3. Start webpack
    ```bash
   npm start
4. Access App
  
   Open a web browser and navigate to http://localhost:3000 to access the app
